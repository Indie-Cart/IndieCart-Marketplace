- Node.js (v18 or higher)

### Setup Instructions
1. Clone the repository
```bash
git clone https://github.com/Indie-Cart/IndieCart-Marketplace.git
```
2. Install Backend Dependencies
```bash
cd Backend
npm install
```
3. Install Frontend Dependencies
```bash
cd ../Frontend
npm install
npm run build
```

3. Start the Application
```bash
cd ..
npm start 
```
